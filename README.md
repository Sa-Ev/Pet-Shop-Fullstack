# Pet Shop Fullstack

## 🛠 Технологии

- **Клиент**: Vite + JavaScript
- **Сервер**: Node.js + Express

## 🚀 Запуск

```bash
# Клиент
cd client && npm install && npm run dev

# Сервер
cd server && npm install && npm start
```

## 📁 Структура

```
Pet-Shop-Fullstack/
├── client/       # Vite-приложение
├── server/       # Node.js API
└── README.md
```
